# Dataset Description

This document describes the dataset structure used in Phase-1 of the project.

The dataset is organized into train, validation, and test splits.
Each split contains identical class folders aligned with the model output labels.

Classes included:
- center
- clean
- donut
- edge_loc
- edge_ring
- loc
- near_full
- scratch
- other

Images are grayscale wafer inspection images curated manually for consistency and balance.
