# Dataset Source

The dataset images are collected from publicly available sources such as:
- Research publications
- Open academic datasets
- Publicly available wafer inspection examples

No proprietary or confidential data is used.

Images are manually reviewed and categorized according to predefined defect classes.

## Dataset Availability

The full dataset used for Phase-1 evaluation is provided as a ZIP archive via an external storage link.
